Quarkus REST API

A modern, reactive REST API built with Quarkus - Supersonic Subatomic Java

Features

Reactive REST API

Built with RESTEasy Reactive for high performance and scalability.

Middleware with Routes

Flexible middleware architecture with declarative routing.

Log Filters

Comprehensive logging system with customizable filters.

Hibernate Reactive

Persistence layer powered by Hibernate Reactive for efficient data management.

Audit Data

Built-in audit trail for tracking data changes and user actions.

Mapstruct

Efficient object mapping with Mapstruct for clean code architecture.

Error Handling

Robust error handling system with detailed error responses.

Swagger Docs

Comprehensive API documentation with Swagger/OpenAPI.

API Documentation

Swagger UI OpenAPI Spec Demo API

Quick Start

# Run the application in dev mode
./mvnw compile quarkus:dev

# Package the application
./mvnw package

# Run the packaged application
java -jar target/quarkus-app/quarkus-run.jar

# Create a native executable
./mvnw package -Dnative
